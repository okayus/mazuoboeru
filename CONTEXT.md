# まず覚える (mazuoboeru)

学んだことをクイズ化して反復で覚える、マルチユーザーの公開学習サービス。
クイズ本体は必ず公開、個人の学習履歴・PII は非公開。

## Language

**Quiz (クイズ)**:
1人のユーザ（作者）が作って公開する、設問の集合体。タイトル、説明、複数の [[Question]] を含む。作成と同時に自動公開される（非公開オプションは存在しない）。
_Avoid_: テスト、試験、問題集

**Session (セッション)**:
ブラウザで Web UI を使う人間ユーザの認証状態。Cookie ベース（HttpOnly + Secure + SameSite）。サーバ側で `session` テーブル管理。
_Avoid_: ログイン情報、認証クッキー

**PAT (Personal Access Token)**:
CLI または AI エージェント（Claude 等）が API を叩くための Bearer トークン。Web の設定画面でユーザが自分で発行し、平文表示は発行直後のみ。失効・スコープを持つ。クイズの量産（AI 駆動の作成）が主な用途。
_Avoid_: API キー、シークレット、アクセスキー、APIトークン（曖昧）

**Publish (公開する)**:
作者が「公開する」ボタンを押す **意図的・不可逆** な遷移。`draft` → `published` の一方向。一度公開したクイズを `draft` に戻すことはできない（[[0002-publish-flow-and-edit-rules]]）。
_Avoid_: リリース、出題、公表（曖昧）

**Draft (下書き)**:
作者本人のみが見える、作成途中のクイズ。タイトルや設問が未完成でも保存できる。`Publish` の対義語的状態。
_Avoid_: 非公開、プライベート（mazuoboeru には private 状態は存在しない）

**Hidden (非表示)**:
モデレータが違反コンテンツに対して付与する状態。一般ユーザからは検索/直リンク共に閲覧不可（403）。作者本人だけ自身の hidden 状態を確認できる。`Draft` とは別概念（draft は作者の意思、hidden は他者の介入）。
_Avoid_: ブロック、削除、停止（曖昧）

**Attempt (挑戦)**:
1人のユーザが1つのクイズに挑む1セッション分の記録。`attempt_answer` を子に持つ。本人のみ可視（公開されない）。
_Avoid_: 回答（answer は設問単位）、トライ、チャレンジ、受験

**Author (作者)**:
クイズを作ったユーザ。公開されるのは [[Display Name]] のみで email は非公開。
_Avoid_: クリエイター、出題者、投稿者

**Display Name (表示名)**:
作者として公開される名前。初回サインインの OAuth プロバイダから取得し、本人が後で編集可能。
_Avoid_: ユーザ名、ニックネーム、ハンドル
